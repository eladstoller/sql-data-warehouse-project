# sql-data-warehouse-project
Building a simplified  modern data warehouse with SQL Server, including ETL processes, data modeling, and analytics.

Project Content:
- Implementing the medallion data Architecture: Bronze, Silver, Gold
- Building an ETL pipeline using SQL
- Data Modeling: creating fact and dimension tables for analytical queries 
- Analytics & Reporting: SQL-based reports for actioanble insights
- Version control using Git and Github (CLI)
